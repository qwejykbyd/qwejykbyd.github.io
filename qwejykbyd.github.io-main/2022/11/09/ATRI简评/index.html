ATRI简评

整部《ATRI》（或者说大部分关于机器人的作品）所要追问的核心问题其实就是：人的本质是什么？或者更精确一些，究竟是什么使得人成为人，而不是一架由蛋白质组成的机器？而又是什么使得机器人不再是机器人，而是一个拥有独立灵魂的个体？

一般来说，这个问题的答案就是人类所独有的情感。这在《ATRI》里被笼统地说成是“心”。如果援引《君与彼女与彼女之恋》中女主角葵的两句台词，也许可以看得更清楚一些：
“
葵……bug了，也可以吗？
会欢笑，会哭泣，会高兴，会难过，会喜欢上你……这样也可以吗？
”
然而，当主角偷看了亚托莉的日志时，他已经将亚托莉的情感解构为单纯的数据了。之前陪伴着主角的是一个活生生的“人”，但是在这之后亚托莉已经是一架纯粹的机器了。事实上即使是人类的头脑也禁不起这样的解构。如果真的将人的头脑看作是神经元之间的电位的话，那么属于人类的一切崇高的东西——生与死、情感与意志、爱与恨、意义与道德——统统都不复存在了。这样的话，人类与亚托莉又有什么区别呢？

因此，《ATRI》真正想要揭示的其实是另一个答案。人的本质并不是人类引以为傲的情感，而是“自由”。亚托莉冷冰冰地说“被夏生先生说喜欢了，我也用喜欢回复了他”，但是人类又何尝不是如此行为的呢？只不过人类与亚托莉的区别在于，人的行为是遵从自己的自由意志的，而亚托莉所遵循的是强加于她之上的命令（比如“要让夏生开心”）。

所以，人之所以成为人，实际上就是因为人对于必然性的命令的违抗。只有人不屈服于自己的命运的时候，人才真正成为了一个人。同样，只有当亚托莉举起机械手臂，将研究员打得奄奄一息的时候，她才真正完成了自我的觉醒。此时的亚托莉再也不是一个合格的机器人了，但是她却成为了一个具有自由意志的、真正的“人”。也只有在完成这场蜕变之后，她才能直面自己对于夏生的情感，确认了自己是因为自己的意志爱上了夏生，而不是因为其他人的命令。

在《创世纪》中，亚当和夏娃偷吃了智慧果，因此被上帝逐出了伊甸园。他们确实真真切切地用自己的行动违抗了上帝的意志，从无尽的必然之中逃脱了出来，决定了自身的命运。也正因为如此，他们才没有沦为上帝的卑微造物，而成为了真正意义上的“人”——尽管这小小的违抗对于上帝是完全不足道的。

我在想，亚托莉是不是也是如此呢？
